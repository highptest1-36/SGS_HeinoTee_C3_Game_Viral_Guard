## Typography

```

Headings (bold, attention-grabbing):
- Font: system-ui, -apple-system, "Segoe UI", Roboto, sans-serif
```

## Visual Effects

### Prompt 9: Card Hover States
```
Design hover and interaction states for feed cards:

Normal state:
- Transform: scale(1)
- Box-shadow: 0 2px 8px rgba(0,0,0,0.1)
- Border: 2px solid transparent

Hover state:
- Transform: scale(1.02)
- Box-shadow: 0 8px 16px rgba(0,0,0,0.2)
- Border: 2px solid #3b82f6 (blue)
- Transition: 0.3s ease

Selected state:
- Border: 3px solid var(--role-color)
- Box-shadow: 0 0 20px rgba(var(--role-color-rgb), 0.5)
- Glow effect

Flagged state:
- Background: rgba(239, 68, 68, 0.1) (red tint)
- Border-left: 4px solid #ef4444

Checked state:
- Background: rgba(59, 130, 246, 0.1) (blue tint)
- Border-left: 4px solid #3b82f6
```

### Prompt 10: Modal Animations
```
Create smooth modal transitions:

Entrance:
- Backdrop: fade in (0.3s ease)
- Modal: slide up from translateY(20px) + fade in (0.3s ease-out)

Exit:
- Modal: fade out (0.2s ease-in)
- Backdrop: fade out (0.2s ease-in, delay 0.1s)

Timing function: cubic-bezier(0.4, 0, 0.2, 1)
```

### Prompt 11: Meter Fill Animation
```
Animate meter bars on value change:

Width transition:
- Duration: 0.5s
- Timing: cubic-bezier(0.4, 0, 0.2, 1)
- Ease: smooth acceleration

Pulse effect (near threshold):
- Animation: pulse 1.5s ease-in-out infinite
- Keyframes: scale 1 → 1.05 → 1
- Opacity: 1 → 0.9 → 1

Gradient movement:
- Background-position animation
- Create "flowing" effect
```

## Accessibility Assets

### Prompt 16: Screen Reader Labels
```
Add ARIA labels and alt text:

Role reveal:
- aria-label="Your role is {role_name}"
- Describe abilities in accessible format

Action buttons:
- aria-label="Share this card to increase viral meter"
- Include consequences in description

Meters:
- aria-valuenow, aria-valuemin, aria-valuemax
- aria-label="Viral meter at {value} out of {threshold}"

Cards:
- aria-label="Card: {title}, credibility {score}%, {flagged/checked status}"
```

## Export Specifications

### Prompt 19: Asset Delivery Format
```
All assets embedded in code:
- Icons: Unicode emojis (no image files)
- Colors: CSS variables in :root
- Fonts: System font stack (no web fonts)
- Effects: Pure CSS animations

Benefits:
- No external dependencies
- Works offline
- Fast loading
- Easy to modify
```

This game uses only code-based assets (CSS, emojis, system fonts) with no external images or icon libraries required. Perfect for hackathon submission and offline play.
